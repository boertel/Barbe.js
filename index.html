Barbe is a Javascript Library that allows you to attach templates to specific elements on your page. It also populates templates from API responses and show automatically a loader during the loading.

Usage

Here is a short example of how to use Barbe in a simple way. Mustache is the templating engine used.

1. Create your template

Html
<script type="text/html" id="my_template" data-anchor="my_anchor"> <p>This is a template building with <em>{{ name }}</em>.</p> </script>

2. Create your anchor in your html file

Html
<div id="my_anchor"></div>

3. Initialize the template

Javascript
new Barbe.View("my_template", { data: {name: "Barbe.js"} }).draw();

4. Result

Html
... <div id="my_anchor"> <p>This is a template building with <em>Barbe.js</em>.</p> </div> ...

To see a full example, check out the github repository: boertel/barbe.js

More Options

Use another templating engine

By default, Barbe is using Mustache but you can use your favorite templating engine such as Hogan or Handlersbar. You can find how to define the compile function for those two, below in the documentation.

Javascript
// Function that compiles the template with data. Barbe.settings.template.engine = Mustache.render; // Type to identify templates. Barbe.settings.template.type = ["text/html"];
Populate a template with an API response

Most of the populated templates use an API response, that's why Barbe provides a easy way to populate the templates. And in bonus, a loader2 is automatically inserted inside the anchor during the API call.

Javascript
var view = new Barbe.View("api_template", { url: "/api/v1/user", success: function (response) { /* ... */ return response; } });

Note: the second object, called provider, is almost the same object that will be passed to jQuery $.ajax function1. The success is overwritten to take the response and use it as data by the templating engine to create the HTML version of the template, so don't forget to return response; at the end of the function.

Execute a function on the data

Mustache does not provide a helper system, but sometimes we need to surcharge the data such as capitalized strings, change the format of numbers.

Javascript
var view = new Barbe.View({template: "api_template", data: [{name: "John"}, {name: "Ben"}], provider: function (response) { for (var i = 0; i < response.length; i++) { response[i].nameUpperCase = response[i].name.toUpperCase(); } return response; } });
